# Elevator Distributed Control System

## 1. 개요
* **목적:** STM32 MCU 간 CAN 통신 기반 엘리베이터 분산 제어 시스템 구현
* **특징:** 메인 컨트롤러(STM32F429)와 층별 노드 컨트롤러(STM32F103) 간 상태 동기화 및 독립적 제어 수행

## 2. 시스템 사양

| 항목 | 상세 규격 |
| :--- | :--- |
| **Main MCU** | STM32F429ZI (메인 로직 연산 및 통신 중계) |
| **Node MCU** | STM32F103C8T6 (BluePill, 층별 제어 노드) |
| **OS / 구조** | CMSIS-OS v2, BSP(Board Support Package) 계층 분리 아키텍처 |
| **통신 인터페이스** | CAN (Standard ID, FIFO0), SW SPI |
| **센서** | TS0224 아날로그 홀센서 (ADC 샘플링 및 캘리브레이션 적용) |
| **액추에이터** | 서보 모터 (TIM PWM 제어, 도어 개폐 구동) |
| **디스플레이** | 0.96인치 OLED (128x64 해상도, Bit-banging SW SPI 적용) |

## 3. 주요 모듈 명세

### 3.1. 통신 모듈 (CAN Communication)
* **수신 버퍼:** `HAL_CAN_RxFifo0MsgPendingCallback` 활용, 크기 8의 순환 큐(Circular Queue) 적용으로 수신 데이터 패킷 유실 방지
* **메시지 라우팅:** 메인 보드의 상태 브로드캐스트 파싱 및 노드 보드의 호출/취소/센서 응답 트랜잭션 처리

### 3.2. 입력 및 센서 (BSP & Sensors)
* **노드 식별:** 하드웨어 핀(PA5~PA7) 전압 상태 기반 물리적 층수(1~3층) 자동 할당
* **자석 감지:** TS0224 센서 ADC 다중 샘플링을 통한 초기 기준값(Baseline) 획득 및 임계값(Threshold) 비교를 통한 감지(Detected) 판별
* **버튼 제어:** 다중 상태 확인(Confirm Count) 및 지연 대기(Delay)를 적용한 소프트웨어 디바운싱(Debouncing) 구현

### 3.3. 디스플레이 제어 (OLED)
* **렌더링 방식:** 브레젠험(Bresenham) 알고리즘 기반 선 긋기 및 폰트 비트맵 확대 출력 로직 적용
* **상태 표출:** 현재 층수 표기, 특수 상태 디코딩 및 이동 방향에 따른 꺾쇠 화살표(Chevron) 프레임 애니메이션 처리

### 3.4. 액추에이터 제어 (Servo Motor)
* **구동 방식:** 타이머 펄스 폭(1260 ~ 2190) 연산 기반 도어 개방(180도) 및 폐쇄(0도) 제어

## 4. CAN 프로토콜 명세

| CAN ID | 송수신 방향 | 데이터 구조 및 기능 |
| :---: | :--- | :--- |
| `0x100` | F429 -> BluePill | 카 위치(층), 이동 방향, 동작 상태(NORMAL/INSP/MOVING) 브로드캐스트 |
| `0x101` | BluePill -> F429 | 탑승객 호출 발생 알림 (UP:1, DOWN:2) |
| `0x102` | BluePill -> F429 | 탑승객 호출 취소 알림 (UP:1, DOWN:2) |
| `0x103` | F429 -> BluePill | 타겟 도착 예정 층에 대한 홀센서 자석 감지 확인 요청 |
| `0x104` | BluePill -> F429 | 타겟 층 홀센서의 자석 감지 완료 응답 (ACK) |
| `0x105` | F429 -> BluePill | 제어 대상 층의 도어 개폐 명령(OPEN:1, CLOSE:2) 및 현재 서비스 방향 전달 |

## 5. 주요 파일 및 디렉토리 구조

| 파일명 | 기능 설명 |
| :--- | :--- |
| `app.c` / `app.h` | 애플리케이션 메인 루프 (초기화, CAN 큐 파싱, UI 업데이트 로직 수행) |
| `bsp.c` / `bsp.h` | 하드웨어 추상화 계층 (보드 간 핀 제어, IO 상태 읽기/쓰기 통합) |
| `can_comm.c` | CAN 필터 초기화, 데이터 Tx/Rx 제어 및 인터럽트 기반 순환 큐 데이터 적재 |
| `ts0224.c` | 홀센서 제어 모듈 (ADC 샘플링, 영점 교정, 감지 여부 판별) |
| `oled_spi.c` | OLED 드라이버 모듈 (SW SPI 제어, 프레임 버퍼 관리, 엘리베이터 UI 렌더링) |
| `servo.c` | 서보 모터 제어 모듈 (목표 각도 입력 기반 PWM 펄스 생성 및 출력) |
| `button.c` | 버튼 입력 모듈 (에지 트리거 감지 및 소프트웨어 필터링) |
| `def.h` | 시스템 전역 상수, CAN ID 및 상태 코드 사전 정의 |
